Quarkus - Community

Getting help

Documentation

We have a lot of documentation, be sure to check our getting started page and all our guides.
Also check out our FAQ section.

Discussions

Stack Overflow

Ask your questions on Stack Overflow.
After the documentation, probably the best place to look for answers. We actively monitor the quarkus tag.

Mailing list

You can also ask questions on our mailing list. Join the Google group via email quarkus-dev+subscribe@googlegroups.com or by visiting the Google Groups page https://groups.google.com/d/forum/quarkus-dev.

Live chat on Zulip

If you prefer live chat with the developers, we have a Zulip chat where we all hang out.

An Open Project

Quarkus is an Open Source project licensed under the Apache License version 2.0. First and foremost, it is an open community where contributions, ideas and discussions are done in the open and welcome contributors. Let’s join forces in building the future of Java applications.

Code and issue tracker

The source code is hosted on GitHub. If you need to report a bug or request a new feature, look for a similar one on our github issues tracker. If you don’t find any, create a new issue.

Contributing

There are a lot of ways to contribute to Quarkus development:

  • Writing documentation and quickstarts,

  • Fixing bugs or even just opening an issue,

  • Writing new extensions.

If you are an extension developer or want to contribute to Quarkus development, join our mailing list either:

  • by emailing quarkus-dev+subscribe@googlegroups.com

  • by visiting the Google Groups page https://groups.google.com/d/forum/quarkus-dev

Make sure you visit our extensive extension developer guide.

Quarkus is open. Quarkus and its extensions are available under the Apache Software License 2.0 or compatible license.
